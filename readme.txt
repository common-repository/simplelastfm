=== simplelastfm ===
Contributors: sadammm
Tags: lastfm, last, fm, simple, xml, simplexml, widget
Requires at least: 2.8
Tested up to: 3.0

Just a simple widget.
Displaying few recent played tracks and most played artists.

== Description ==

Plugin is using simplexml to connect with last.fm's xml files.
No password required - you only need to provide your username.

== Installation ==

Upload the simplelastfm plugin to your blog, Activate it.
Configure it just like other widgets.
